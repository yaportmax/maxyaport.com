# maxyaport.com

Minimal Astro site for writing, projects, and an about page.

## Commands

```sh
npm install
npm run dev
npm run build
```

## Writing

Copy `src/content/writing/_template.md`, rename it, set `draft: false`, and write the post body.

## Projects

Copy `src/content/projects/_template.md`, rename it, set `draft: false`, and add project details.

## Secrets

Porkbun API keys are stored in `.env.local`. Do not commit that file.
